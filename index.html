CSS Cheat Sheet

by Cameron Guay

Step One is to Link

To link your css file to the html file, use <link href='.styles.css' type='text/css' rel='stylesheet'>

CSS Selectors

Name Tag Description
Type Selector h1{} These select an HTML element.
Universal Selector *{} This is used to select all elements.
Class Selector .box{} This is used, combined with a period, to select class elements.
id Selector #unique This is used, combined with a '#', to select id elements.
Attribute Selector a[title]{} This selects attributes based on a given definition.
Pseudo-class Selector p:first-child{} This selects elements during certain circumstances.
Pseudo-element Selector p::first-line{} This selects a specific part of an element.
Descendant Combinator article p This will only select an element when the first selector is the parent of the second selector.
Child Combinator article > p This works similar to Descendant, however the second selector must be the direct child.
Adjacent Sibling Combinator h1+p This selects the second selector when both selectors have the same parent and the second selector is immediately following the first selector.
General Sibling Combinator h1~p This selects the second selector when both selectors have the same parent. Doesn't need to immediatly follow first selector.
